# rc-nest-api

## Description

This Repo is about the nest framework.

## Links

- Source Repo: https://github.com/rohith0696/rc-nest-api
- Hosted on Heroku: https://rc-nest-api.herokuapp.com/

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
